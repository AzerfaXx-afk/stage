# StageFlow 🏛️

> Journal de stage interactif pour le projet IoT — Ville de Luxembourg

## Fonctionnalités

- 📊 **Dashboard** — Progression du stage, stats en temps réel
- 📝 **Journal** — Entrées quotidiennes avec filtres et recherche
- 🎙️ **Reconnaissance vocale** — Dicte tes entrées en français
- 🎯 **Missions** — 34 tâches organisées en 5 phases
- 📤 **Export** — Rapport Markdown, backup JSON, résumé hebdo
- 📱 **PWA** — Installable sur mobile comme une app native
- 💾 **Persistance** — Données sauvegardées en localStorage

## Stack

- [Next.js 15](https://nextjs.org) — React framework
- [Framer Motion](https://www.framer.com/motion/) — Animations
- [Web Speech API](https://developer.mozilla.org/en-US/docs/Web/API/Web_Speech_API) — Voix
- CSS Variables — Design system custom

## Lancer en local

```bash
npm install
npm run dev
```

## Déployer

```bash
npm run build
```

Le dossier `out/` contient le site statique prêt à déployer.
